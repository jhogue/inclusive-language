Inclusive Language Checker

A tool to check strings of text for exclusive language, highlight them, and suggest alternate language that is more inclusive.

Copy and paste text into the Input box, click the button to submit, and then review the suggestions.

  • Exclusive words/phrases count
  • Word count
  • Sentence count
  • Paragraph count
  1. When text is scanned for exclusive words or phrases, additional information will display in this location for further reading and education about the topics highlighted.

By J. Hogue as part of the DesignXRI Equity program. Many thanks to all language suggestion sources. Contribute to the project on GitHub.
